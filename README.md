# DL1
You shuold use python version 3.8 -> 3.11 because tensorflow is support from version 3.8 to 3.11
Download DATASET from https://www.kaggle.com/datasets/techsash/waste-classification-data?resource=download

To running this project, following step by step:
1.  Python -m venv venv.
2.  Run file .\venv\Scripts\Activate.ps1 in powershell.
3.  Run pip install -r requirements.txt to install all the library need for run code.
4.  If your code is error at import library its reason could be you were not choose correct Python Interpreter. You can use Ctrl + Shift + P --> Python: Select Interpreter --> Python 3.xx (venv).
5.  Run file train.py first to train model.
6.  Then run file test_model.py to test model anh predict from pictures.
